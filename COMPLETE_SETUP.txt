ğŸ”§ SECTION 7: TROUBLESHOOTING

Problem: "Repository not found" when pushing
Solution:
1. Check GitHub username is correct
2. Make sure repository is PUBLIC
3. Verify repository name is correct

Problem: Website not appearing
Solution:
1. Wait 3-5 minutes after deployment
2. Go to Settings â†’ Pages to verify status
3. Check that files were uploaded successfully
4. Try refreshing browser (Ctrl+Shift+R)

Problem: Images not showing
Solution:
1. Make sure image files are uploaded to GitHub
2. Check file names match exactly (case-sensitive)
3. Use correct relative paths in HTML
4. Re-upload image files if needed

Problem: Contact form not working
Solution:
1. Clear browser cache
2. Try in different browser
3. Make sure email client is configured
4. Check contact.html is properly uploaded

Problem: Mobile menu not working
Solution:
1. Clear cache and refresh
2. Try different browser
3. Make sure script.js is uploaded
4. Check for JavaScript errors in console
